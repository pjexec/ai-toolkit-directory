# AI Toolkit Directory

Personal searchable directory of AI tools with ratings and capability matching.

## Features
- Search by task or capability (e.g., "image to video", "voice receptionist")
- Filter by category, minimum rating, or primary tools only
- Click stars to adjust ratings
- Status tags show primary/active/secondary/abandoned tools

## Setup

```bash
npm install
npm run dev
```

## Built with
- React 18
- Vite
- Tailwind CSS
